#### String.prototype.includes(searchString[,position])
> 判断某一字符串是否包含另一个字符串

参数
- searchString：要搜索的字符串
- position：从当前字符串的第 position 个位置开始搜索，默认为 0

返回值
- 如果包含，返回 true
- 如果不包含，返回 false
